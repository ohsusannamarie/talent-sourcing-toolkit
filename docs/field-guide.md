## 🧠 A few opinions about recruiting technology

Technology does not replace recruiter judgment.

The best recruiters are not the people with the most tools. They're the people who:

- ask better questions
- understand talent markets deeply
- communicate clearly
- recognize patterns
- build trust quickly
- think in systems instead of transactions

AI will absolutely change recruiting. But the recruiters who win long-term will probably
look less like "prompt engineers" and more like:

- talent strategists
- workflow architects
- market interpreters
- relationship builders
- information synthesizers

Tools matter. But thinking matters more.

## 👋 If you're brand new to sourcing

Do not try to learn everything at once. Seriously.

Start with:

1. Google X-ray searching
2. Boolean basics
3. LinkedIn Recruiter
4. Outreach fundamentals
5. One sourcing workflow you can repeat consistently

You do not need:

- 75 Chrome extensions
- an AI agent army
- seven enrichment platforms
- a second brain built in twelve apps

Most great sourcing is still curiosity, persistence, pattern recognition, communication,
and knowing where to look. Learn the fundamentals first. Then layer on tooling intentionally.

## 🏷️ Tagging & filtering (now live)

The library is fully tagged and searchable. Every resource carries lightweight metadata
that powers the [searchable site](https://ohsusannamarie.github.io/talent-sourcing-toolkit/):

- **Pricing** — free / freemium / paid
- **Region** — global / EU / APAC
- **Status** — active / acquired
- **Curation** — community favourite (❤️), new in 2026
- **Format** — for conferences: IRL / virtual / hybrid

Filter by any combination, search across all 364 entries, deep-link to a filtered view,
and compare tools side by side. This is the "less like a static list, more like a
searchable recruiting knowledge system" goal — shipped.

**Caution flags are live too.** Entries that scrape platforms, surface personal contact
data, or use AI to screen candidates carry a ⚠ badge — a thing-to-consider, not an
accusation. See the [disclosure & caution policy](disclosure.md) for what each flag
means and what to check.

Next on the metadata roadmap: topical tags (`#technical-recruiting`, `#osint`,
`#automation`, `#talent-intelligence`) for finer filtering.

## 🌐 Long-term vision

This started as a bookmark collection. It's gradually becoming:

- a recruiting field guide
- a sourcing operating system
- a talent intelligence ecosystem map
- a recruiter enablement resource
- a modern reference library for how recruiting actually works now

The recruiting profession is changing fast. AI, automation, labor market intelligence,
workflow tooling, and sourcing infrastructure are becoming core recruiter skills — not
side hobbies. The goal of this repo is to help recruiters adapt *intentionally* instead
of drowning in disconnected tools and endless noise.

Some of these tools are enterprise-grade. Some are weird little internet goblin utilities
held together with caffeine and hope. Both are useful.
