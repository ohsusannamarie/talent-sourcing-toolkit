# Resource Library

A curated library of recruiting, sourcing, talent intelligence, AI, productivity, and recruiter enablement resources.

Use this as the big browseable layer of the Talent Sourcing Command Center. The main README explains the system. This page holds the resource inventory.

> **Note:** Tools, pricing, availability, ownership, and compliance posture change constantly. Validate before using anything in an active recruiting workflow.

#### Legend

💰 = no free or freemium version clearly available

❤️ = especially useful or high-signal resource
