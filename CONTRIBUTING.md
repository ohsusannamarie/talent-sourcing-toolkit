# Contributing to Talent Sourcing Toolkit

Thanks for helping make this resource more useful.

This repo is a curated sourcing resource library, not a dumping ground for every recruiting tool on the internet. The goal is quality, usefulness, and practical context.

---

## Good contributions

Useful contributions include:

- New sourcing tools
- Google Custom Search Engines
- Chrome extensions relevant to sourcing or research
- Recruiter productivity tools
- Dead-link fixes
- Category/tag suggestions
- Security or privacy concerns
- Notes about deprecated or abandoned tools
- Better descriptions or use cases

---

## Before suggesting a tool

Please check:

- Does it still exist?
- Is it relevant to sourcing, recruiting, talent intelligence, or research?
- Is it reasonably safe to use?
- Does it request risky permissions?
- Is the creator/vendor identifiable?
- Would a recruiter understand why it matters?

If the answer is "technically yes, but spiritually no," maybe don't submit it. We are trying to reduce chaos, not become chaos.

---

## Suggested submission format

When opening an issue, include:

```md
## Tool or resource name

## Link

## Category

## What it helps sourcers do

## Recommended use case

## Caveats, risks, or notes

## Is this free, paid, freemium, or unknown?
```

---

## Security and privacy notes

Please flag anything that:

- Requests broad browser permissions
- Reads or changes data on websites
- Touches email, LinkedIn, cookies, contacts, or candidate data
- Has unclear ownership
- Has not been updated in a long time
- Appears abandoned
- Has questionable privacy practices

A sourcing tool can be clever and still not be worth the risk.

---

## Tone and style

Keep it practical, plainspoken, and useful. Descriptions should help a recruiter understand what the tool does and when to use it.

Avoid vendor fluff. If a description sounds like it was written by a SaaS homepage generator after drinking six cold brews, please simplify it.
