# Contributing to Talent Sourcing Field Guide

Thanks for helping make this resource more useful.

This repo is a curated sourcing field guide, not a dumping ground for every recruiting tool on the internet. The goal is quality, usefulness, current context, and practical workflow value.

---

## Good contributions

Useful contributions include:

- New sourcing tools
- Google Custom Search Engines
- Chrome extensions relevant to sourcing or research
- Recruiter productivity tools
- AI recruiting tools and responsible-use notes
- Talent intelligence resources
- Market research resources
- Workflow recipes
- Templates and checklists
- Dead-link fixes
- Category and tag suggestions
- Security or privacy concerns
- Notes about deprecated, acquired, abandoned, or risky tools
- Better descriptions or practical use cases

---

## Before suggesting a tool or resource

Please check:

- Does it still exist?
- Is it relevant to sourcing, recruiting, talent intelligence, recruiting operations, or hiring research?
- Is it reasonably safe to use?
- Does it request risky permissions?
- Is the creator or vendor identifiable?
- Would a recruiter understand why it matters?
- Is there a clear use case beyond “this seems neat”?
- Could it create privacy, AI, compliance, or data-handling concerns?

If the answer is “technically yes, but spiritually no,” maybe do not submit it. We are trying to reduce chaos, not become chaos.

---

## Suggested submission format

When opening an issue or pull request, include:

```md
## Tool or resource name

## Link

## Category

## Best for

## Recruiting use case

## Free, paid, freemium, or unknown?

## Status
Active / Needs Review / Deprecated / Experimental

## Compliance-sensitive?
Yes / No / Unsure

## Caveats, risks, or notes
```

## Resource status legend

| Status | Meaning |
|---|---|
| ✅ Active | Currently useful and available |
| 🟡 Needs review | May need updated pricing, features, ownership, or availability |
| ⚠️ Compliance-sensitive | Use carefully; may involve privacy, AI, browser permissions, or personal data concerns |
| ❌ Deprecated | No longer active, outdated, unsafe, or not recommended |
| 🧪 Experimental | Interesting, but not battle-tested |

---

## Security and privacy notes

Please flag anything that:

- Requests broad browser permissions
- Reads or changes data on websites
- Touches email, LinkedIn, cookies, contacts, or personal data
- Uses AI to score, rank, screen, or recommend candidates
- Has unclear ownership
- Has not been updated in a long time
- Appears abandoned
- Has questionable privacy practices

A sourcing tool can be clever and still not be worth the risk.

---

## Tone and style

Keep it practical, plainspoken, and useful. Descriptions should help a recruiter understand what the tool does, when to use it, and what to watch out for.

Avoid vendor fluff. If a description sounds like it was written by a SaaS homepage generator after drinking six cold brews, please simplify it.
