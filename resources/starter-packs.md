# Starter Packs

The point of this toolkit is not to use everything. The point is to build a sane, useful stack for the sourcing problem in front of you.

Start with one pack. Test a few resources. Keep what earns its spot.

---

## New Sourcer Starter Pack

For sourcers who are learning the basics or building their first research workflow.

### Search the archive for

- `profile`
- `resume`
- `LinkedIn`
- `boolean`
- `people search`
- `contact`
- `email`
- `tabs`
- `notes`

### Build a small stack

| Stack piece | Goal |
|---|---|
| 2-3 people/profile search resources | Find public candidate profiles faster |
| 1-2 resume or portfolio search resources | Practice X-ray and keyword search |
| 1 notes or clipping tool | Capture findings without tab chaos |
| 1 browser/tab workflow helper | Keep research sessions manageable |
| 1 data cleanup helper | Clean copied lists or exported research |

### Watch out for

Do not install every shiny Chrome extension. Start boring. Boring is underrated. Boring usually does not request permission to read your entire browser life.

## OSINT-Adjacent Research Stack

For deeper public web research while staying ethical, careful, and grounded.

### Search the archive for

- `OSINT`
- `public records`
- `social`
- `profile`
- `metadata`
- `image`
- `username`
- `domain`
- `company`
- `research`

### Build a small stack

| Stack piece | Goal |
|---|---|
| Public profile search | Find publicly available professional signals |
| Username/domain research | Connect public web breadcrumbs carefully |
| Metadata/context tools | Validate what you find before acting on it |
| Documentation workflow | Record what is known, unknown, and inferred |
| Ethics checklist | Avoid creepy, invasive, or non-job-relevant research |

### Watch out for

Just because something is technically findable does not mean it belongs in a recruiting workflow. Keep research job-relevant, public, proportionate, and respectful.

---

## My strong recommendation

Build one stack at a time.

A good sourcing stack should feel like a sharp little toolkit, not a junk drawer with a Chrome icon.
